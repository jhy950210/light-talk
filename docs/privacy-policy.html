2. 개인정보의 처리 및 보호

  • 전화번호 비식별화: 원본 전화번호는 서버에 저장하지 않습니다. HMAC-SHA256 알고리즘으로 생성된 해시값(Blind Index)만 저장하여 원본 복원이 불가능합니다.
  • 비밀번호 암호화: BCrypt 단방향 해시 알고리즘으로 암호화하여 저장합니다.
  • 통신 암호화: 모든 데이터 전송 시 SSL/TLS(HTTPS) 암호화를 적용합니다.

3. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, SMS 인증을 위해 아래 서비스를 이용합니다:

제공받는 자 제공 항목 제공 목적
Solapi (주식회사 쿨에스엠에스) 휴대전화번호 SMS 인증번호 발송

4. 개인정보의 파기

  • 회원 탈퇴 시 관련 개인정보를 지체 없이 파기합니다.
  • SMS 인증번호(OTP)는 발급 후 5분 경과 시 자동 파기됩니다.
  • 인증 토큰은 발급 후 10분 경과 시 자동 파기됩니다.
