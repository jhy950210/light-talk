라이트톡은 가볍고 빠른 실시간 채팅 앱입니다. 복잡한 기능 없이, 소중한 사람들과 빠르게 대화하세요.

주요 기능

- 실시간 메시지: 친구와 즉시 대화하세요. 메시지가 실시간으로 전달됩니다.
- 사진 & 동영상 공유: 대화 중 사진과 동영상을 간편하게 보내고 받으세요.
- 간편한 가입: 전화번호 인증만으로 빠르게 시작할 수 있습니다. 이메일이나 복잡한 절차가 필요 없습니다.
- 닉네임#태그 시스템: 고유한 닉네임#태그로 친구를 쉽게 찾고 추가하세요.
- 친구 관리: 닉네임으로 검색하고, 친구 요청을 보내고 받으세요.

라이트톡은 빠르고 가벼운 채팅 경험에 집중합니다. 지금 시작하세요.
